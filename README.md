An editorial investigation mapping the chronological distributions, research categories, and deep migration metrics of Nobel Laureates from the Indian subcontinent and its global diaspora. 

### 🌟 Key Architecture & Layout Highlights
* **The Chronological Matrix:** A full-width wide-canvas scatter grid plotting breakthroughs using custom emoji text-layer coordinates, minimizing standard visualization noise.
* **On-Demand Profile Inspector:** A customized information layout block pulling biographical data and cached public-domain image files instantly upon selection.
* **Dual-Track Segmentations:** Compares localized, domestic policy/humanitarian wins directly against capital-intensive international hard science pipelines.
